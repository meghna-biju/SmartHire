# SmartHire

A full-stack job application tracker built with Spring Boot, React, JWT Authentication, H2/MySQL, and Tailwind CSS.

## Features

- JWT Authentication
- User Registration & Login
- Add/Edit Job Applications
- Search & Filter Jobs
- Interview Notes
- Status Management
- Analytics Dashboard

## Tech Stack

Frontend:
- React
- React Router
- Axios
- Tailwind CSS

Backend:
- Spring Boot
- Spring Security
- Spring Data JPA
- JWT
- Hibernate

Database:
- H2
- MySQL (production)
- 
## Getting Started

Backend

```bash
cd backend
mvn spring-boot:run
```
Frontend
```
cd smarthire-frontend
npm install
npm run dev
```
